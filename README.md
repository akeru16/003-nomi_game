# のみげーむ (Nomi Game)

## 概要 (Overview)
「のみげーむ」は、日本中のユーザーが飲みゲーを投稿・共有できる掲示板プラットフォームです。
「企業があまりできなさそうなこと」をテーマに、誰もが自由に楽しめる参加型サイトを目指します。

## 主な機能 (Key Features)

### 1. ユーザー投稿機能 (User Generated Content)
- アカウント作成をしたユーザーなら誰でも、オリジナルの飲みゲーを投稿可能。
- 飲み会の場で生まれたローカルなゲームを日本中に広めることができます。

### 2. ランキング機能 (Rankings)
- **人気ランキング**: ユーザーからの評価や閲覧数に基づいた人気の飲みゲーを表示。
- **新着ランキング**: 新しく投稿された飲みゲーをリアルタイムでチェック。

### 3. お助け機能・スマホサポート (Digital Support)
- **ランダム選出**: 何をして遊ぶか迷った時に、ランダムで飲みゲーを提案。
- **ゲームサポートツール**: デフォルトで収録されているゲーム用に、スマホを活用した補助機能（サイコロ、タイマーなど）を提供。

## コンセプト (Concept)
- 企業が参入しづらいニッチで自由な領域。
- 誰でも気軽に参加・発信できるオープンなコミュニティ。

## 技術スタック (Tech Stack)
- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
